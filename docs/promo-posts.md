## Reddit / Discord Post

I built an Android client for LM Studio called LMSMOB Chat.

The idea is simple: run LM Studio on your PC, start the local server, then use
your Android phone as the chat client over your LAN.

What it supports:

- LM Studio OpenAI-compatible endpoint at `/v1`
- Native LM Studio `/api/v1/chat` for server tools
- MCP/plugin integrations configured in LM Studio
- Chat history with search and session switching
- Copy/edit user messages
- JSON import/export for chat history
- Image attachments for vision-capable models
- Emulator default URL and physical-device LAN setup

Repo:
https://github.com/mindylab/lmsmob_chat
